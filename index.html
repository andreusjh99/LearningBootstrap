Colours

There are a few default bootstrap colours. You can use them in any element by just specifying the classes:
  • bg-* for background
  • text-bg-* for background with text colour optimised for the background
  • text-* for text
  • border-* for border

Note that bg-* does not set color property. Use text-bg-* for that.

Note also that for all the colours, you can get a lighter version by adding subtle to the end

Spacings

Margins and paddings

Notation: {property}{sides}-{breakpoint}-{size}
Eg. mx-sm-4
Note that {sides} and {breakpoint} are not compulsory. Blank for sides mean all 4 sides. Blank for breakpoint means the xs breakpoint.
  • property is either m for margin or p for padding.
  • sides is one of:
    • t - for top (margin-top/padding-top)
    • b - for bottom (margin-bottom/padding-bottom)
    • s - for start, i.e. left for LTR, right for RTL
    • e - for end, i.e. right for LTR, left for RTL
    • x - for both left and right
    • y - for both top and bottom
  • size can be: 0, 1, 2, 3, 4, 5, auto

Borders

To have a border around an element using BS, just specify either one of these classes:
  • border
  • border-top
  • border-bottom
  • border-start
  • border-end

Specifying border-*-0 will take away the border from that side.

Width

Use border-{nunber}, where number can be 0, 1, 2, 3, 4, or 5.

Rounded corners

Use rounded-{number}, where number can be 0, 1, 2, 3, 4, or 5.
